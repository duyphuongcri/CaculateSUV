Standardized Uptake Values (SUV)

To aid in the interpretation of PET images, the Pixel Value tool can be used to view the absorption of an injected isotope in body tissue, based on Standardized Uptake Values (SUV). SUV statistics are indicated based on the DICOM data present in the image, and on the user preferences for SUV display (body weight, ideal body weight, lean body mass, and body surface area). For details on configuring user preferences for SUV data display, see Standardized Uptake Value (SUV) Plugin.

Standard Uptake Values are not absolute values. For details on the equations used to calculate these values in the application, see SUV Equations and References.

The registration of Standardized Uptake Values is reported each time the Pixel Value tool is applied over an area of the viewport, along with the pixel position (row, column) in the image. The value of a given pixel is expressed in BQML (Becquerels/milliliter)

To use the Pixel Value tool:

1. From the Image Manipulation toolbar, expand the Window/Level toolset and select the Pixel Value tool.

2. Click and drag the tool over regions of a PET image. Standardized Uptake Values are displayed whenever the pixel values change under the application of this tool. The values disappear when the mouse button is released.

Note: If the Pixel Value tool is used to measure a value that is outside the limits of a dataset, or the value cannot be determined, the pixel value is reported as Value = Unknown. The Standardized Uptake Value(s) displayed (e.g., Body Surface Area [BSA], Body Weight [BW], Ideal Body Weight [IBW], and Lean Body Mass [LBM]) are based on a user preference. For details, see Standardized Uptake Value (SUV) Plugin.

Standardized Uptake Values are displayed only if the image contains values for the required DICOM tags. For details, see Requirements for Displaying Standardized Uptake Values.
